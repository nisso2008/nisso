# Weather Diary (Дневник погоды)

**Автор:** Рахимова Бахриниссо 

## Описание
Приложение для ведения дневника погоды с графическим интерфейсом (Tkinter).  
Позволяет:
- Добавлять записи с датой, температурой, описанием и осадками
- Просматривать все записи в таблице
- Фильтровать записи по дате и температуре
- Сохранять и загружать данные в/из JSON

## Как запустить
1. Убедитесь, что установлен Python 3.6+
2. Сохраните код в файл `weather_diary.py`
3. Запустите:
   ```bash
   python weather_diary.py
